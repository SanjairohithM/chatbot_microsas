Features

  • Chat Icon: A floating chat button that appears in the bottom-right corner
  • Center Modal: Chat opens in the center of the screen, not in the button space
  • Overlay Background: Semi-transparent background when chat is open
  • Responsive: Adapts to different screen sizes
  • Customizable: Colors and appearance can be customized via theme parameters
  • Easy Integration: Simple iframe embed code

Usage Instructions

  1. Replace 1 in the iframe src with your actual bot ID
  2. Customize the theme colors by modifying the theme parameter
  3. Adjust the iframe width and height as needed for your layout
  4. Test on both desktop and mobile devices
